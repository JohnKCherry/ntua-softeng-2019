Οι προγραμματιστές έχουν να καλύψουν τις ανάγκες των υπόλοιπων Stakeholders. Δυστυχώς, δεν είναι τόσο εφικτό να υπάρχει κάποιο σύστημα τεχνητής νοημοσύνης που να βλέπει ύποπτες κινήσεις είτε από επιχειρήσεις (π.χ. μεγάλες αυξομειώσεις τιμών από τις επιχειρήσεις ή αγορές μεγάλων ποσοτήτων από χρήστες)	. Μία άλλη διαδικασία που δεν θα μπορούσε να επιτευχθεί είναι άμεσα είναι ένα σύστημα αξιολόγησης χρηστών που θα μπορούσε να βοηθήσει τους developers να μπορούν να αναγνωρίσουν τους κακόβουλους χρήστες.
