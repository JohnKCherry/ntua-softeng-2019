Οι προγραμματιστές απαιτούν να έχουν μια γρήγορη και πλήρως λειτουργική εφαρμογή η οποία θα μπορεί να είναι ελκυστική προς τους χρήστες. Σημαντικό είναι να υπάρχει άμεση αλληλεπίδραση με τις άλλες κλάσεις των Stakeholders ώστε να γίνονται άμεσα αλλαγές για να γίνει ακόμα πιο θελκτική στους χρήστες, εφόσον όσο περισσότεροι θα είναι οι χρήστες που θα κάνουν αγορές τόσο πιο πολλές θα είναι οι επιχειρήσεις που θα θέλουν να συνεργαστούν μαζί τους, έτσι θα έχουν μεγαλύτερο κέρδος.
