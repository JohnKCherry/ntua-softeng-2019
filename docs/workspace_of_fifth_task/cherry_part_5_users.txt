Ένας χρήστης θα απαιτούσε μια εφαρμογή η οποία να έχει φιλικό interface. Δυνατότητες για λίστα επιθυμιών (wishlist), όπως και δυνατότητες σύγκρισης προϊόντων. Άλλες σημαντικές λεπτομέρειες που είναι πιο εξειδικευμένες. Ένα από αυτά είναι συστήματα τεχνητής νοημοσύνης και νευρωνικής μάθησης, ώστε όχι μόνο να βγάζει τα δυνατότερα δυνατά προτεινόμενα, αλλά σύμφωνα με τις εποχές να παίρνει σαν παραμέτρους τις εξωτερικές μεταβλητές του περιβάλλοντος. Τέλος, μια τεχνολογία που θα μπορούσε να χρησιμοποιηθεί είναι οι ειδοποιήσεις του browser, ώστε να τους ενημερώνει για προσφορές, ενημερώσεις κτλ.
